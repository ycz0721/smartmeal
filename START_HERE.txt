步骤1：配置环境变量
  cp .env.example .env
  
  然后编辑 .env 文件，填入：
  - DEEPSEEK_API_KEY（必填，从 https://platform.deepseek.com 获取）
  - NEXTAUTH_SECRET（运行 openssl rand -base64 32 生成）
  - DATABASE_URL（PostgreSQL 连接字符串）

步骤2：安装依赖并初始化数据库
  npm install
  npx prisma migrate dev --name init
  npx prisma db seed

步骤3：启动开发服务器
  npm run dev
  
  访问 http://localhost:3000
